# EVENT-HANLDER
📚 Student Activity Manager
💻 Description

A React-based app to manage student activities.
You can add students, toggle details, mark attendance, and delete records — all updated instantly in React.
No backend or storage is used.

🧩 Features

➕ Add Students

👁️ Toggle Details

✅ Mark Present / ❌ Mark Absent

🗑️ Delete Students

📊 Dashboard: Total, Present, Absent

🧱 Components

App.jsx – main state and functions

StudentForm.jsx – add students using refs

StudentList.jsx – render all student cards

StudentCard.jsx – display student info with buttons

🎯 React Concepts

useState – manage students

useRef – focus input field

Event Handlers – click & submit

Passing arguments – to handle specific students

Conditional rendering – show/hide details

List rendering with map()
